Configuración con Cadena de Conexión

Ideal para desarrollo y acceso directo a datos existentes

settings.json (VS Code) 
{
  "mcpServers": {
    "MongoDB": {
      "command": "npx",
      "args": [
        "-y",
        "mongodb-mcp-server",
        "--connectionString",
        "mongodb://localhost:27017/myDatabase",
        "--readOnly"
      ]
    }
  }
}
Cursor Configuration 
{
  "mcpServers": {
    "MongoDB": {
      "command": "npx",
      "args": ["-y", "mongodb-mcp-server"],
      "env": {
        "MDB_MCP_CONNECTION_STRING": "mongodb://localhost:27017/myDatabase",
        "MDB_MCP_READ_ONLY": "true"
      }
    }
  }
}
claude_desktop_config.json 
{
  "mcpServers": {
    "MongoDB": {
      "command": "npx",
      "args": [
        "-y",
        "mongodb-mcp-server"
      ],
      "env": {
        "MDB_MCP_CONNECTION_STRING": "mongodb+srv://user:pass@cluster.mongodb.net/db",
        "MDB_MCP_READ_ONLY": "true"
      }
    }
  }
}
Comando CLI 
npx -y mongodb-mcp-server@latest \
  --connectionString="mongodb://localhost:27017/myDatabase" \
  --readOnly

Configuración con API de Atlas

Para gestión completa de infraestructura y aprovisionamiento de clústeres

Paso 1: Crear Service Account

1
Acceder a Atlas

Ve a cloud.mongodb.com → Access Manager → Organization Access

2
Crear Service Account

Add New → Applications → Service Accounts

Asigna solo los permisos mínimos necesarios (ver tabla de permisos)

3
Guardar Credenciales

Copia el Client ID y Client Secret inmediatamente

El Secret no se muestra de nuevo
settings.json (VS Code) 
{
  "mcpServers": {
    "MongoDB": {
      "command": "npx",
      "args": [
        "-y",
        "mongodb-mcp-server",
        "--apiClientId",
        "your-client-id",
        "--apiClientSecret",
        "your-client-secret",
        "--readOnly"
      ]
    }
  }
}

Configuración con Docker

Entorno aislado sin necesidad de instalación local de Node.js

Docker con Connection String 
docker run --rm -i \
  -e MDB_MCP_CONNECTION_STRING="mongodb+srv://user:pass@cluster.mongodb.net/db" \
  -e MDB_MCP_READ_ONLY="true" \
  mongodb/mongodb-mcp-server:latest
Docker con Atlas API 
docker run --rm -i \
  -e MDB_MCP_API_CLIENT_ID="your-client-id" \
  -e MDB_MCP_API_CLIENT_SECRET="your-client-secret" \
  -e MDB_MCP_READ_ONLY="true" \
  mongodb/mongodb-mcp-server:latest
Docker en archivo MCP 
{
  "mcpServers": {
    "MongoDB": {
      "command": "docker",
      "args": [
        "run",
        "--rm",
        "-i",
        "-e",
        "MDB_MCP_CONNECTION_STRING=mongodb+srv://user:pass@cluster.mongodb.net/db",
        "-e",
        "MDB_MCP_READ_ONLY=true",
        "mongodb/mongodb-mcp-server:latest"
      ]
    }
  }
}

Opciones de Configuración Avanzada

Variables de Entorno

Configuración via ENV 
# Credenciales
export MDB_MCP_CONNECTION_STRING="mongodb://..."
export MDB_MCP_API_CLIENT_ID="client-id"
export MDB_MCP_API_CLIENT_SECRET="client-secret"

# Configuración
export MDB_MCP_READ_ONLY="true"
export MDB_MCP_INDEX_CHECK="true"
export MDB_MCP_LOG_PATH="/path/to/logs"

Transporte HTTP

Para conexiones remotas o integración web

Solo para desarrollo
Servidor HTTP 
npx -y mongodb-mcp-server@latest \
  --transport http \
  --httpHost=127.0.0.1 \
  --httpPort=3000

Referencia Completa de Opciones

Opción CLI Variable ENV Descripción Default
--connectionString MDB_MCP_CONNECTION_STRING URI de conexión a MongoDB -
--apiClientId MDB_MCP_API_CLIENT_ID Client ID de Service Account Atlas -
--readOnly MDB_MCP_READ_ONLY Solo operaciones de lectura y metadatos false
--indexCheck MDB_MCP_INDEX_CHECK Forzar uso de índices en consultas false
--disabledTools MDB_MCP_DISABLED_TOOLS Lista de herramientas a desactivar -
--transport MDB_MCP_TRANSPORT Protocolo de transporte (stdio/http) stdio

Configuración de Seguridad

Modo Solo Lectura

Restringe el servidor a operaciones de solo lectura, previniendo modificaciones accidentales

--readOnly
Recomendado por defecto

Control de Herramientas

Desactiva herramientas específicas o categorías completas según necesidades

--disabledTools create delete atlas
Mínimo privilegio

Verificación de Índices

Obliga a usar índices en todas las consultas, rechaza collection scans

--indexCheck
Rendimiento

Permisos Atlas Recomendados

Casos de Uso Básicos

Org Member Listar organizaciones/proyectos
Project Read Only Ver clústeres existentes
Project Cluster Manager Crear/gestionar clústeres

Gestión Administrativa

IP Access List Admin Gestionar listas de acceso
Database Access Admin Gestionar usuarios de BD
Organization Owner ❌ Evitar - Riesgo de seguridad
