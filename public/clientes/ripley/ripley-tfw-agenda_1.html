Fundamentos de Arquitectura y Rendimiento

Del Modelo Key-Value al Modelo Documental

  • Análisis y disección del product.json real
  • Reto EAV: Indexar eficientemente el array attributes
  • Impacto de anidación en related y parentpricestock
  • Definición de fronteras MongoDB vs ElasticSearch

Patrones de Diseño de Esquema

  • Attribute Pattern: Transformación del array attributes para filtros (Marca + Color + Talla)
  • Extended Reference: ¿Duplicar info del shop (Seller)?
  • Anti-patrones: Índices en arrays que crecen infinitamente

Estrategia de Índices para Latencia <10ms

  • Indexación Compuesta y Multikey para queries profundas
  • Facetas y conteos para navegación lateral
  • Gestión del Working Set en RAM (M60)
  • Query Optimization con .explain()
db.products.find({"related.price.sale.value": {$lt: 50000}})

Alta Disponibilidad y Resiliencia

  • Replica Sets: Failover automático <5 segundos vs Sentinel
  • Topología Multi-Zona: GCP us-central1 (Zonas A, B, C)
  • Tolerancia a caídas de zona completa
02
45 minutos Enfoque: Conectividad con Microservicios y Kafka

Integración y Ecosistema de Datos

Arquitectura Event-Driven con Kafka

  • Kafka Sink Connector: Ingesta simple de eventos
  • Atlas Stream Processing: Validar y transformar JSON en vuelo
  • Caso de Uso: Detectar updates de stock sin reescribir documento completo
  • Change Streams: Notificar cambios de precio en tiempo real

Best Practices de Conectividad

  • Connection Pool para alta concurrencia
  • Manejo de Timeouts y Reintentos (Retryable Writes/Reads)
  • Lecturas Secundarias: readPreference primario vs secundarios
  • Soporte: NodeJS, Go, Python
{ readPreference: "secondaryPreferred", maxPoolSize: 100 }
03
45 minutos Enfoque: Preparación para el CyberDay

Operaciones Críticas y Escalabilidad

Escalabilidad para Picos de Tráfico

  • Auto-Scaling: Reglas para escalar CPU y Disco automáticamente
  • Tiering: Subir de M60 a M80 durante eventos (Zero-Downtime)
  • Estrategia de escalado pre y post CyberDay

Observabilidad y Monitoreo

  • Performance Advisor: Identificación proactiva de queries lentas
  • Alertas Clave: IOPS, Tickets, Replication Lag
  • Dashboards personalizados para el equipo

Seguridad y Red

  • VPC Peering: Conexión privada con GCP
  • Gestión de Accesos: Usuarios de BD vs IAM Roles
  • Encryption at rest y in transit
