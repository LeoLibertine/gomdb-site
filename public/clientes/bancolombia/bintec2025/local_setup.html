📓 Jupyter Notebook - Setup Completo

1

Instalar Python

Python 3.8 o superior es requerido

1. Descarga desde: python.org/downloads

⚠️ IMPORTANTE: Marca "Add Python to PATH" durante la instalación

# Verificar instalación python --version
# Con Homebrew (recomendado) brew install python # Verificar python3 --version
# Ubuntu/Debian sudo apt update sudo apt install python3 python3-pip # Verificar python3 --version
2

Crear Entorno Virtual

Aísla las dependencias de tu proyecto

# Crear carpeta del proyecto mkdir fraud-detection cd fraud-detection # Crear entorno virtual python -m venv venv # Activar (Windows) venv\Scripts\activate # Activar (Mac/Linux) source venv/bin/activate

✅ Verás (venv) al inicio de tu terminal cuando esté activo

3

Instalar Dependencias

# Instalar todas las librerías necesarias pip install pymongo sentence-transformers jupyter notebook ipywidgets plotly # Si da error, instala una por una: pip install pymongo pip install sentence-transformers pip install jupyter pip install notebook pip install ipywidgets pip install plotly
4

Habilitar Widgets

Necesario para el dashboard interactivo (celda 19)

# Habilitar extensiones jupyter nbextension enable --py widgetsnbextension
5

Descargar Notebook

1. Ve a tu Google Colab

2. File → Download → Download .ipynb

3. Guarda en tu carpeta fraud-detection

6

Iniciar Jupyter

# Desde la carpeta del proyecto con venv activo jupyter notebook

✅ Se abrirá automáticamente en: http://localhost:8888

7

Ejecutar Notebook

1. Click en fraud_detection.ipynb

2. Ejecuta las celdas en orden (Shift + Enter)

⚠️ En la celda 4, ingresa tu USER_PREFIX cuando te lo pida

💻 VS Code + Jupyter Extension

1

Instalar VS Code

Descarga desde: code.visualstudio.com

2

Instalar Extensiones

En VS Code, busca e instala:

  • ✅ Python (Microsoft)
  • ✅ Jupyter (Microsoft)
  • ✅ Pylance (Microsoft)
3

Configurar Proyecto

# Terminal en VS Code (Ctrl+`) python -m venv venv # Activar (Windows) venv\Scripts\activate # Activar (Mac/Linux) source venv/bin/activate # Instalar dependencias pip install pymongo sentence-transformers ipykernel ipywidgets plotly
4

Seleccionar Kernel

1. Abre el notebook en VS Code

2. Click en "Select Kernel" (esquina superior derecha)

3. Elige "Python Environments..."

4. Selecciona el entorno venv

🐛 Solución de Problemas

❌ Error: "ModuleNotFoundError: No module named 'pymongo'"

El módulo no está instalado o el entorno virtual no está activo

✅ Solución:
# Verifica que venv esté activo (deberías ver (venv) en la terminal) # Luego reinstala pip install pymongo
❌ Error: "Connection timeout" con MongoDB

No puedes conectarte a MongoDB Atlas

✅ Solución:
  • Verifica que tu IP esté en el whitelist de MongoDB Atlas
  • Revisa que el connection string sea correcto
  • Asegúrate de tener conexión a internet
❌ La celda 3 se queda cargando (asterisco *)

El modelo sentence-transformers está descargándose

✅ Solución:

Dale 2-5 minutos. Si sigue sin cargar, usa un modelo más ligero:

# Reemplaza en celda 3: model = SentenceTransformer('paraphrase-MiniLM-L3-v2') # Más ligero
❌ Los widgets no aparecen (celda 19)

Las extensiones de ipywidgets no están habilitadas

✅ Solución:
# Reinstalar y habilitar pip install --upgrade ipywidgets jupyter nbextension enable --py widgetsnbextension # Reinicia Jupyter después
